Steps to be followed for compilation and Execution of assignmentP2.java

Step 1 : Please copy the file assignmentP2.java in your directory along with the makefile.txt and input.txt from sannunc1_java_p2.tar.gz folder

Step 2 : It is important that there exists an input.txt file in the same directory where assignmentP2.java,makefile.txt is present.If it is not there then please create one named input.txt

Step 3 : Type the command make -f makefile.txt



Step 3 will  compile the code ,Execute the code as well as create and open a text file named output.txt in the same directory

The OUTPUT is in the following format
<PROBLEM_NUMBER> <PROBLEM_SIZE> <NUMBER_NODES_ADDED> <MIN_COST> <PATH> <RUNTIME>
